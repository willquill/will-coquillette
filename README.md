# How I Built This

`nvm install 12`

`nvm use 12`

`npm install @lekoarts/gatsby-theme-minimal-blog`

`npm install -g gatsby-cli`

`gatsby new will-coquillette LekoArts/gatsby-starter-minimal-blog`

`cd will-coquillette`

Finally, look at the links below to customize it for your own use.
